# 闲鱼智能客服 Chrome 扩展 — 界面设计文档

## 一、界面概览

扩展包含三个主要界面：

| 界面 | 入口 | 用途 | 尺寸 |
|------|------|------|------|
| **Popup 快捷面板** | 点击浏览器工具栏扩展图标 | 查看运行状态、快速开关功能 | 380 × 500 px |
| **Options 设置页** | 右键扩展图标 → 选项 / 点击 Popup 中的"详细设置" | 完整配置 AI、通知、话术 | 自适应（全宽） |
| **Content 侧边栏** | 注入到闲鱼页面右侧 | 类似微信网页版的对话管理面板 | 300 px 宽（可收起） |

- **设置按钮**：跳转到 Options 完整配置页
- **查看完整记录**：展开更长的对话历史面板

## 四、Content 侧边栏设计

注入到闲鱼页面右侧，形态类似微信网页版侧边栏。默认吸附在页面右边缘，不遮挡主聊天区，支持展开、收起与宽度记忆。

### 展开状态

```
┌──────────────────────────────────────────────────────────────────────┬──────────────────────────────┐
│ 闲鱼原页面聊天区                                                     │ 🐟 智能客服侧边栏            │
│                                                                      ├──────────────────────────────┤
│                                                                      │ 运行状态  ● 在线             │
│                                                                      │ 今日回复  12                 │
│                                                                      │ 今日订单  3                  │
│                                                                      ├──────────────────────────────┤
│                                                                      │ 当前会话  买家A              │
│                                                                      │ 商品      iPhone 15 Pro      │
│                                                                      ├──────────────────────────────┤
│                                                                      │ 买家消息                      │
│                                                                      │ 还能便宜吗                   │
│                                                                      │                              │
│                                                                      │ AI 建议回复                  │

│                                                                      │ 喜欢的话可以直接拍下         │
│                                                                      │                              │

│                                                                      │ 快捷操作                     │
```

### 收起状态

```
┌──────────────────────────────────────────────────────────────────────┬────┐
│ 闲鱼原页面聊天区                                                     │ 🐟 │
│                                                                      │ 2  │
└──────────────────────────────────────────────────────────────────────┴────┘
```

### 布局说明

- **右侧固定栏**：宽度建议 280 到 320 px，固定在页面右侧，类似微信网页版联系人详情栏
- **顶部状态区**：显示扩展在线状态、今日回复数、订单数、当前模式
- **当前会话区**：展示当前买家、当前商品、最近一条消息、AI 建议回复
- **操作区**：包含发送、编辑、忽略、暂停自动回复等高频按钮
- **最近会话区**：列出待处理、已回复、新订单等关键会话，便于快速切换
- **收起形态**：收起后保留窄条入口，显示未处理数量红点

### 交互说明

- **审核模式**：当开启审核模式时，AI 回复先展示在侧边栏的当前会话区，用户点击发送后才真正发出
- **会话切换**：点击最近会话区中的买家项，可切换当前分析对象，并同步显示对应建议回复
- **快捷操作**：支持一键插入常用话术、订单提醒、商品链接等操作
- **暂停按钮**：可对当前会话或全局暂停自动回复
- **宽度记忆**：用户调整过的侧边栏宽度与展开状态写入 [`chrome.storage`](plans/xianyu-chrome-extension-plan.md) 持久化保存

## 六、视觉风格

- **主色调**：闲鱼品牌色（#FF6B00 橙色）
- **辅助色**：
  - 成功/开启：#52C41A（绿色）
  - 警告：#FAAD14（黄色）
  - 错误/关闭：#F5222D（红色）
- **字体**：系统默认字体栈，中文优先 PingFang SC / Microsoft YaHei
- **圆角**：8px 统一圆角
- **阴影**：`0 2px 8px rgba(0,0,0,0.15)`

---

## 七、响应式适配

- **Popup**：固定 380×500，内容溢出时内部滚动
- **Options**：自适应宽度，最小 800px，左侧导航固定 200px
- **Content 侧边栏**：固定吸附右侧，宽度支持用户调整并记住偏好；当页面宽度不足时自动切换为窄栏模式
