隐私政策

最后更新日期：2026年3月27日

「闲鱼智能客服助手」（以下简称"本扩展"）尊重并保护用户的隐私。本隐私政策说明本扩展如何收集、使用和保护您的信息。

1. 信息收集与使用

本扩展在运行过程中会处理以下信息：

  • 闲鱼聊天消息：读取买家发送的消息内容，用于生成 AI 回复。仅在用户明确授权自动直发后，系统才会自动向买家发送回复。消息仅在本地处理和临时存储，不会上传至本扩展开发者的服务器。
  • 商品信息：读取当前会话关联的商品标题，用于提供更准确的回复。
  • 用户配置：包括 AI 服务设置、通知设置、知识库内容等，存储在 Chrome 浏览器本地存储中（chrome.storage）。

2. 第三方服务

本扩展需要调用以下第三方 AI 服务来生成回复，您的消息内容会被发送至您所选择的 AI 服务商：

  • DeepSeek（https://api.deepseek.com）
  • OpenAI（https://api.openai.com）
  • 阿里云通义千问（https://dashscope.aliyuncs.com）
  • 或您自行配置的其他 AI 服务

请注意，这些第三方服务有各自的隐私政策，建议您阅读相关条款。

3. 通知服务

如果您启用了订单通知功能，订单相关信息（商品名称、金额）会通过您配置的通知渠道发送：

  • 浏览器通知（本地，不涉及第三方）
  • 钉钉机器人（您配置的 Webhook 地址）
  • 飞书机器人（您配置的 Webhook 地址）
  • Telegram Bot（https://api.telegram.org）

4. 数据存储

本扩展不设有独立服务器，不会将您的数据上传到本扩展开发者的任何服务器。
  • 所有配置数据存储在 Chrome 浏览器的本地存储中（chrome.storage.sync / chrome.storage.local）。
  • 消息日志仅保存在本地，用于您在扩展界面中查看历史记录。
  • API Key 等敏感信息仅存储在浏览器本地，不会被传输至本扩展开发者。

5. Cookie 使用

本扩展读取闲鱼网站的登录 Cookie（如 _m_h5_tk），仅用于：

  • 验证您的闲鱼登录状态
  • 通过闲鱼官方 API 获取消息和发送回复

本扩展不会将您的 Cookie 传输至任何第三方。

6. 权限说明

  • storage：存储扩展配置和消息日志
  • notifications：发送浏览器桌面通知
  • tabs：与闲鱼页面通信，发送和接收消息
  • cookies：检测闲鱼登录状态
  • scripting：在闲鱼页面注入消息监听脚本
  • sidePanel：提供侧边栏操作面板

7. 数据安全

本扩展采取以下措施保护您的数据安全：

  • 所有与 AI 服务的通信均使用 HTTPS 加密传输
  • 不在日志中记录完整的 API Key
  • 仅在必要时访问闲鱼页面数据

8. 用户权利

您可以随时：

  • 在扩展设置页面查看和修改所有配置
  • 通过"重置所有设置"清除所有本地数据
  • 卸载扩展以删除所有相关数据

9. 政策更新

本隐私政策可能会不时更新。更新后的政策将在本页面发布，并更新"最后更新日期"。
